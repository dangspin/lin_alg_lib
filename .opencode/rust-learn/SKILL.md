# Rust Learn

这个技能旨在通过数值分析和线性代数的实际案例，引导用户深入学习 Rust。

## 激活方式
当用户输入以下类似指令时激活：
- "帮我生成一个topic"
- "学习下一个topic"
- "我想练习 Rust 数值计算"

## 工作流程
1. **挑选主题**：挑选一个符合当前项目背景（数值分析或线性代数）且难度适中的 Rust 学习主题。
2. **提供代码模版**：
   - 仅包含函数签名（Function Signatures）。
   - **严禁** 直接编写 `struct` 或 `enum` 的实现，但可以提供相关的设计提示（Hints）。
   - 用户负责完成函数体（Implementation）和结构定义。
3. **提供 Example 模版**：
   - 提供一个完整的、可运行的示例文件（通常位于 `examples/` 目录下）。
   - 示例代码应展示如何使用即将实现的功能。
